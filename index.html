Strange Attractors

Strange Attractors tend to have beautiful, complex, fractal structures that are highly dependent on starting factors. This experiment implements a number of attractors, including Clifford, Fractal Dream, and Hopalong (my personal favorite).

The input parameters of the attractors are interactive and the attractor can be visualized as a series of points or the path generated by the points over time.

Conway's Game of Life

Conway's Game of Life is a 2-dimensional cellular automata simulation invented by John Conway in 1970. It has 4 simple rules that govern the behavior of the automata:

  1. Any live cell with fewer than two live neighbors dies, as if by underpopulation.
  2. Any live cell with two or three live neighbors lives on to the next generation.
  3. Any live cell with more than 3 live neighbors dies, as if by overpopulation.
  4. Any dead cell with exactly three live neighbors becomes a live cell, as if by reproduction.

These simple rules create unexpected, emergent behavior. Keep a look out for the different patterns - still lifes (stable patterns), oscillators (periodic repeating patterns), and spaceships (traveling patterns).

The game runs itself and is randomly initialized on every page load.
